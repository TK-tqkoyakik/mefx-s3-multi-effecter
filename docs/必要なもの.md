| 必要なもの | 用途 | 状態 |

| Windows PC | 開発作業 | 使用中 |

詳しくは [部品購入チェックリスト](checklists/procurement.md) を見てください。

## 部品の買い方の注意

PCM1808とPCM5102Aは、最初はIC単体ではなくモジュール/ブレイクアウト基板を買ってください。

理由:

- IC単体は周辺部品と細かいはんだ付けが必要
- モジュールなら `BCK`、`LRCK`、`DIN/DOUT`、`VCC`、`GND` のようなピンラベルを見ながら配線できる
- 今回の配線図はモジュール前提で作っている

## 測定に必要なもの

| 道具 | 目的 |
| --- | --- |
| テスター | 9V/5V/3.3V/4.5V確認 |
| ブレッドボードまたはユニバーサル基板 | 試作 |
| ジャンパーワイヤ | 仮配線 |
| シールド線 | 音声信号のノイズ低減 |
| 小型アンプまたはオーディオIF | 出力音確認 |

オシロスコープやロジックアナライザがあるとI2S確認が楽ですが、最初は必須ではありません。
