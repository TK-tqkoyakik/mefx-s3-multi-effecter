# 部品購入チェックリスト

最初からケースや基板発注まで買わず、まず机上で動作確認できる部品だけを買います。

## 先に買うもの

- [ ] ESP32-S3開発ボード
- [ ] PCM1808 ADCモジュール、または変換基板
- [ ] PCM5102A DACモジュール
- [ ] TL072オペアンプ
- [ ] 6.35mmモノラルジャック x2
- [ ] モーメンタリーフットスイッチ
- [ ] 9Vエフェクター用電源
- [ ] 9V DCジャック
- [ ] 5Vレギュレーターモジュール
- [ ] 3.3Vレギュレーターモジュール
- [ ] 小型マイクモジュール
- [ ] ブレッドボード、またはユニバーサル基板
- [ ] ジャンパーワイヤ
- [ ] シールド線
- [ ] 1MΩ、100kΩ、10kΩ、4.7kΩを含む抵抗セット
- [ ] 100nF、1uF、10uF、47uFを含むコンデンサセット
- [ ] 逆接保護用ダイオード、または保護モジュール
- [ ] テスター

## 後で買うもの

机上テストで音が出てから買います。

- [ ] 金属エフェクターケース
- [ ] 最終基板、またはPCB発注
- [ ] 絶縁タイプの6.35mmジャック
- [ ] LED
- [ ] ノブ
- [ ] 配線固定具
- [ ] 絶縁材

## 買う前に確認すること

- [ ] ESP32-S3開発ボードのGPIOが使いやすい配置になっている
- [ ] PCM1808とPCM5102AがI2S対応である
- [ ] 9V電源の極性を確認した
- [ ] TL072を単電源9Vで使うため、4.5V仮想GNDを作る予定がある
- [ ] まだケース加工やPCB発注をしない

## 作業を止める条件

以下の場合は、次の部品購入や組み立てに進まないでください。

- [ ] 3.3Vが安定しない
- [ ] 5Vが安定しない
- [ ] 4.5V仮想GNDが作れていない
- [ ] TL072の出力がすぐクリップする
- [ ] ESP32が起動しない
