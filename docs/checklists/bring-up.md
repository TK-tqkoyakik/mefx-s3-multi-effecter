# 組み立て確認チェックリスト

新しい配線や基板を作ったら、必ずこの順番で確認します。

## 1. 電源だけ確認する

- [ ] ESP32、PCM1808、PCM5102A、TL072をまだ接続しない
- [ ] 9Vを入れる
- [ ] 逆接保護後の電圧を測る
- [ ] 5Vを測る
- [ ] 3.3Vを測る
- [ ] 4.5V仮想GNDを測る
- [ ] 電源を切って、発熱や異臭がないか確認する

## 2. アナログ入力を確認する

- [ ] TL072を接続する
- [ ] 入力ジャックを接続する
- [ ] TL072の電源ピンを測る
- [ ] 入力バイアスが4.5V付近になっているか確認する
- [ ] 小さいテスト信号、またはギター/ベースを入れる
- [ ] TL072出力が4.5V中心で動いているか確認する
- [ ] 強く弾いてもすぐにクリップしないか確認する

## 3. デジタル音声を確認する

- [ ] PCM1808をESP32のI2S入力ピンへ接続する
- [ ] PCM5102AをESP32のI2S出力ピンへ接続する
- [ ] ESP32にファームウェアを書き込む
- [ ] 起動ログに `MEFX-S3 firmware started` が出る
- [ ] 可能ならBCLK/LRCLKをオシロスコープやロジックアナライザで確認する

## 4. 出力を確認する

- [ ] PCM5102A出力が入力信号に反応する
- [ ] 出力バッファ後のDCオフセットが大きくない
- [ ] アンプの音量を最小から少しずつ上げる
- [ ] バイパス、または低エフェクト設定でクリーン音が出る

## 5. フットスイッチを確認する

- [ ] 短押しでログが出る
- [ ] 長押しでバイパスが切り替わる
- [ ] スイッチを押してもESP32がリセットしない
