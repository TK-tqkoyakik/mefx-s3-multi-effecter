# 実体配線図 v1

このページは、実際にブレッドボード/ユニバーサル基板で組むための配線指示です。

## 前提

- ESP32-S3は開発ボードを使います。
- PCM1808とPCM5102Aは、まずはIC単体ではなくモジュール/ブレイクアウト基板を使います。
- モジュールのピン名は製品によって少し違うため、同じ意味のラベルへ読み替えてください。
- 最初はケースに入れず、机上で動作確認します。

## 2. TL072のピン

DIP-8のTL072を上から見て、くぼみ/丸印を上にした時のピン番号です。

```text
      TL072 DIP-8 上面
       _________
 OUT A |1     8| +9V
 IN- A |2     7| OUT B
 IN+ A |3     6| IN- B
  GND  |4     5| IN+ B
       ---------
```

この試作では:

- A側: 入力プリアンプ
- B側: 出力バッファ

## 組む順番

1. 電源だけ作る。
2. 9V、5V、3.3V、4.5Vをテスターで測る。
3. TL072入力側だけ作る。
4. TL072出力が4.5V中心で動くか見る。
5. ESP32、PCM1808、PCM5102Aをつなぐ。
6. Webシミュレーターでプリセット操作を確認する。
7. ESP32実機を接続してファームウェアを書き込む。
8. 小音量で出力音を確認する。

## 絶対に確認してから次へ進むこと

- [ ] 9Vレールが正しい
- [ ] 5Vレールが正しい
- [ ] 3.3Vレールが正しい
- [ ] VGND_4V5が約4.5V
- [ ] TL072 pin8が+9V、pin4がGND
- [ ] ESP32とADC/DACのGNDが共通
- [ ] 出力ジャックTipに大きなDC電圧が出ていない
