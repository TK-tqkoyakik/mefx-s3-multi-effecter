# PCでWeb Bluetooth接続する手順

この手順は、スマホではなくPC上のChromeまたはEdgeからMEFX-S3へ接続するためのものです。

## 使うブラウザ

- 推奨: PC版 Google Chrome
- 代替: PC版 Microsoft Edge
- 非対応: Firefox、通常のSafari

Web Bluetoothは `localhost` または `https` のページでのみ使えます。
このプロジェクトの開発URL `http://localhost:5173/` は使用できます。

## 接続前チェック

1. ESP32-S3にファームウェアを書き込む。
2. ESP32-S3をUSBまたは9V電源で起動する。
3. PCのBluetoothをオンにする。
4. Webアプリを `http://localhost:5173/` で開く。
5. アプリの「PC Bluetooth接続」欄を見る。

## 接続する

1. `実機に接続` を押す。
2. ブラウザのBluetooth選択画面が開く。
3. `MEFX-S3` を選ぶ。
4. 接続後、表示が `実機接続` と `GATTサービス準備完了` になる。
5. `MEFX-S3へ送信` でプリセットを送る。

## シミュレーターとの違い

- `実機に接続`: PCのBluetoothでESP32-S3へ接続する。
- `シミュレーター`: ESP32なしで画面と保存/送信の流れだけ確認する。

今のアプリではこの2つを分離しています。
URLに `?sim=1` が付いていても、`実機に接続` はBluetooth選択画面を開きます。

## 接続画面が出ないとき

- ChromeまたはEdgeで開いているか確認する。
- URLが `localhost` になっているか確認する。
- PCのBluetoothがオンか確認する。
- ESP32-S3が起動しているか確認する。
- ESP32-S3のファームウェアがBLE対応版か確認する。
- すでに別のPC/スマホに接続されている場合は、そちらを切断する。

## 接続できた後に確認すること

- 本体プリセット一覧の `更新` が動く。
- 手動エディターで値を変えて `MEFX-S3へ送信` できる。
- 本体プリセットを `読み込み` でエディターへ戻せる。
- 本体プリセットを削除できる。
- ステータス欄に入力/出力/マイクレベルが表示される。
